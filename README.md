# Admin Starter

TypeScript monorepo implementing Better Auth with [API](./apps/api/) and [web](./apps/web/) apps.

## Features

- Registration / Login
- Admin list users

## Run

```sh
docker compose up -d
```

Database files are stored in the `./data` folder. Open http://localhost:3000 to access the UI.

## Stack

- Better Auth
- Biome
- Bun
- Hono
- Lucide
- PostgreSQL
- React
- Tailwind
- TanStack Form
- TanStack Query
- TanStack Start
- TanStack Table
- Vite
- Zod
